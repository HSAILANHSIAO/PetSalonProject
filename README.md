# PetSalonProject

## 寵物沙龍預約表
-  製作動機:
*  a- 提供客戶友善的預約表,可新增預約和修改預約
*  b-用MFA的方式進行客戶驗證
*  C-供客戶選擇寵物的服務內容
-  如下:
1. **預約表**  
   可在此進行預約設定。

2. **商品區**  
   提供各種寵物商品建檔修改功能。

3. **基本資料**  
   用戶管理基本信息，寵物資料。

---

### 安裝套件 maven api + html：

#### Backend（後端）：
- **Spring Boot 3.0**  
- **MyBatis 3.0**  
- **MySQL 8.0**  

#### Frontend（前端）：
- **Vue 3.0**  
- **Ajax**  
- **Query 套件**  
- **HTML**
